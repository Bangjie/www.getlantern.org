How Does Lantern Work?

Lantern is an Internet proxy tool. It’s goal is access: creating fast, reliable connections to the open Internet. Lantern uses secure servers and peer access points to provide Internet access to blocked sites. Lantern is unique because it uses peer connections as a source of Internet connectivity when servers are unavailable.
